Math:
expr			: term MUL|DIV term
term			: factor PLUS|MINUS factor
factor 			: MINUS INT|FLOAT

Vars:
declaration		: INT IDENTIFIER
initialization	: INT IDENTIFIER EQUALS expr
assignment		: IDENTIFIER EQUALS expr


condition		: IF ( CONDITION ) :
					BLOCK
				  ELSE IF( CONDITION ):
				  	BLOCK
				  ELSE:
				  	BLOCK
